# 序章：競馬オッズのモデリングにおける本質的課題とアプローチ

## 1. 競馬のオッズとは何か？
競馬における「オッズ（倍率）」とは、JRA（日本中央競馬会）があらかじめ「この馬が強い」と設定している固定の倍率ではありません。それは**「市場参加者（競馬ファン）の支持率（資金の投票行動）」**の結果として形成される、一種の金融市場システム（パリミュチュエル方式）に基づく変動相場です。

したがって、機械学習モデルを用いてオッズを予測するということは、純粋に「どの馬が強いか」という能力の絶対評価を行うのではなく、**「大衆が過去のデータ、血統、直前情報などからどの馬を強いと信じ、どこに最も資金を投下するか」**を数学的に推測する、高度な「群衆心理のモデリング」に他なりません。

## 2. ツールとアプローチを選定した経緯
競馬のオッズ生成エンジンの精度を向上させるためには、単に「勝敗」を予測するだけでなく、**「市場の群衆心理」と「競走馬の潜在能力」**の両面からアプローチする必要があるという課題認識からスタートしました。当初のデータセットと今後の拡張を見据え、以下の要素を段階的にモデルへ導入するアプローチを選定しました。

### 競走能力と環境要素の定量化
* **タイム指数とラップ構成**: 馬場状態（良〜不良）の違いによる時計の補正を加えたスピード指数や、上がり3ハロン（末脚）の速さに加えて道中のペース（ハイ・スロー）適性。
* **コース適性とトラックバイアス**: 右・左回り、直線の長さ、坂の有無、そして当日のレース進行に伴う「内・外の有利不利」の動的更新。
* **人間系と当日の状態**: 騎手・調教師のコース別勝率や陣営の相性、外厩（育成牧場）仕上げの影響、当日の馬体重増やパドックでのパフォーマンス低下の数値化。
* **市場流動性と乖離**: 締め切り直前の大口異常投票の検知や、AIの予測勝率と市場の支持率（オッズ）との乖離（期待値）の算出。

### 実行タスクとモデルの選定
これらの複雑な非線形要素を統合し、「ファンがなぜそのオッズを形成したか」を逆算するためには、以下のサイクル実行と**アンサンブル学習/モデルの分離**が不可欠であると判断しました。
1. **特徴量エンジニアリングの深化**: 確定単勝オッズを目的変数とし、ファンがどの要素を重視したかを回帰分析で特定。
2. **モデルの分離統合**: 新馬戦（データ少）、重賞（情報多）、ダートなど条件ごとに最適なアルゴリズム（**LightGBM, XGBoost, Deep Learning**など）を選択し、スタッキングで統合。
3. **直近モデルの選定**: 開発の第一歩として、テーブルデータにおいてカテゴリ変数や欠損値の扱いに優れ、高速かつ高精度な勾配ブースティング決定木である **LightGBM** を中核アーキテクチャに据えました。

### 精度の評価指標 (Metrics)
評価には単なる的中率（Accuracy）ではなく、オッズ妥当性を徹底して測るため以下を導入しています。
* **対数損失（Log Loss）**: 予測確率が実際の着順結果とどれだけ乖離していたかのペナルティ評価（$LogLoss = -\frac{1}{N}\sum\sum y_{ij}\log(p_{ij})$）。
* **回収率シミュレーション**: 生成オッズに基づき「期待値 > 1.0」となる馬を買い続けた場合の収支曲線の安定性の確認。

## 3. JRAの「単勝払戻率（控除率）」の基本ルール
日本の競馬（JRA）において、勝馬投票券（馬券）の各種別には、競馬法で定められた「払戻率（還元率）」が設定されています。参加者の購入総額からJRAの収益（控除率）を差し引いた残りのシェアが、的中者に分配される仕組みです。
このシステムは「パリミュチュエル方式」と呼ばれています。

* **参考URL**: [パリミュチュエル方式（Wikipedia）](https://ja.wikipedia.org/wiki/%E3%83%91%E3%83%AA%E3%83%9F%E3%83%A5%E3%83%81%E3%83%A5%E3%82%A8%E3%83%AB%E6%96%B9%E5%BC%8F)

単勝（1着になる馬を当てる馬券）の場合、この払戻率は**約80%**（$R=0.8$）に設定されています。

* **参考URL（公式）**: [JRA公式FAQ - 勝馬投票券の払戻金計算方法](https://jra.jp/faq/faq/f01.html#02_02)
* **参考URL（公式）**: [JRA公式 - 各種馬券の払戻率](https://www.jra.go.jp/keiba/baken/information/haitou/)

## 4. 数学的配分モデル
ある馬 $i$ への投票支持率（全体票数に対する得票割合）を $P_i$ と定義します。
JRAの基本計算式によれば、当該馬のオッズ $O_i$ は以下のパリミュチュエル方式の基本式で完全に記述することができます。

$$O_i = \frac{R}{\sum_{j=1}^{n} P_j} 	imes \frac{1}{P_i} = \frac{0.8}{P_i}$$

ここで、全出走馬の支持率の合計は理論上 $\sum P_i = R = 0.8$ となるよう事前調整されています（残りの20%＝0.2はJRAの運営費や国庫納付金となるため）。つまり、オッズが2.0倍の馬は、市場全体の40%の支持（$0.8 / 2.0 = 0.4$）を得ていることを意味します。

## 5. プロジェクトの最終目標
本プロジェクトの核心的な目標は、前述のアプローチと評価指標に基づき、各レースにおける各馬の支持率 $P_i$ を極めて高い精度で推定することです。支持率を完全にモデリングできれば、その逆数をとることで「レース発走前の確定オッズ」をシステム上で完全にシミュレートすることが可能になります。
