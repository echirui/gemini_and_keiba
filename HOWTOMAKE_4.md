# 第四段階：人間要素と市場心理・バイアスの深層モデリング (Cycle 8-10)

本章では、物理的・客観的な能力指標だけでは説明がつかない「オッズの歪み」を捉えるため、競馬ファンの群衆心理や行動経済学的なバイアスを数値化してモデルに組み込んだプロセスを解説します。

競馬においてオッズは強さの絶対指標ではなく、「前走はどうだったか」「誰が乗るのか」といった定性的な情報・印象に強く左右されます。

## 1. Cycle 8: 人馬の相性（ゴールデンコンビ実績）
どれほど優秀なトップジョッキーであっても、馬の気性や脚質と手が合わなければ実力を引き出せません。逆に、マイナーな騎手でも特定の馬にだけは完璧に騎乗する「主戦騎手」の概念が競馬には存在します。

* **`combo_avg_finish_pos`** : その騎手と特定の馬のペアにおける、過去の平均着順の履歴スコア。

過去に何度もコンビを組んで好成績を挙げている「勝ち上がりコンビ」が再結成された場合、ファンはそれを「勝負気配が高い陣営の意欲」と受け取り、過剰に資金を投下する傾向があります。このコンビ実績変数はそうしたファンの安心感（支持率の上昇）を捉えるのに機能しました。

## 2. Cycle 9: 乗り替わりのペナルティと期待
騎手の変更（乗り替わり）は競馬において極めて重要なシグナルです。

* **`jockey_changed`（乗り替わりフラグ）** : 「前走と同じ騎手（継続騎乗＝0）」か、「前走から別の騎手へ変更（乗り替わり＝1）」かを示すバイナリ変数。

競馬ファンは一般的に、前走好走した時に同じ騎手から依頼を断られず「継続騎乗」となっている状態を「陣営が手放さないほど勝負気配が高い」と評価し、単勝が売れやすい心理的バイアスを持っています。逆に不可解なタイミングでの乗り替わりはオッズを下落させます。モデルはこのフラグを通じて、ファンのシグナル解釈を学習しました。

## 3. Cycle 10: 前走の「新聞上の印象」と実績の格付け
競馬ファンが馬券を買う際、最も参考にし、最もオッズに直結するのが「競馬新聞に印字されている直近のレース（前走）の成績表の文字面」です。本質的な強さよりも、前走の着順がオッズを決定づける巨大な要因となります。

* **`prev_finish_pos`** : 前走の着順（大敗していると大幅に人気を落とす）
* **`prev_popularity`** : 前走での人気順（前走人気だった馬は、今回も人気を引き継ぎやすい）
* **`prev_odds`** : 前走のオッズ
* **`horse_is_g1_performer`** : 過去にG1（最高峰レース）へ出走・好走・格付けされたことがあるかのフラグ（いわゆる「実績上位馬・格上の概念」）

これらの「前哨戦の印象」に関するパラメータを組み込んだことで、モデルの予測精度は飛躍しました。「本質的なスピード指数などは劣るが、前走でたまたま展開に恵まれて勝っており、かつ新聞上の見栄えは良いため、今回は過剰に人気してオッズが下がるだろう」という、馬の真の能力とは無関係の「群衆の心理的挙動に基づくオッズ変動」を高精度で模倣し始めたのです。
