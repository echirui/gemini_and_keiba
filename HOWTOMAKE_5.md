# 第五段階：対数領域への写像とキャリブレーション問題 (Cycle 11-13)

この段階に至ることで、線形的な予測精度は一つの限界（アッパーバウンド）に到達しつつありました。それを根本から打破し、実用レベルのAIとしての堅牢性を獲得したのが、数学的モデリング空間の抜本的改革です。実装としては `train_v13_final.py` への布石となった重要なサイクルです。

## 1. Cycle 11: 対数支持率 (Log-Probability) による学習空間の変換
支持率 $P$ という変数は構造上、 $0 < P < 0.8$ の極めて狭い範囲に全て押し込められています。圧倒的な大本命馬であっても $P = 0.5$（単勝1.6倍）であり、大穴馬（単勝200倍）に至っては $P = 0.004$ のような極端に微小な少数値をとります。

元の線形領域（そのままの支持率）でLightGBMの木構造を構築すると、これらの「0の周りの微小な差分（0.01と0.005の差がもたらすオッズの変動の大きさ）」を効果的に分岐の条件として切り分けることができませんでした。

そこで、目的変数を **「支持率の自然対数空間（Logarithmic Space）」** へと再変換し、その空間上で学習させるアーキテクチャに変更しました。

$$\log(y_{train}) = \log\left(\frac{0.8}{Odds_{actual}}\right)$$
$$ \log( \hat{P}\_i ) = f(Features\_i) $$

値のスケールが、負の大きな極値（大穴）からゼロに近い浅い負の値（本命）へと劇的に広がり、対数領域でMSE（平均二乗誤差）の最適化が可能になりました。
（※予測推論プロセスでは、最後に $\exp(\text{予測値})$ で対数ベクトルを元の実数確率空間に戻します。）
この空間の変更により、大穴馬（オッズ100倍を150倍と予測してしまうブレ）と本命馬の確率における桁数の違いをアルゴリズムが学習できるようになり、予測精度が革命的に向上しました。

## 2. Cycle 12-13: 数学的キャリブレーションと本命・大穴バイアス
競馬市場をはじめとする不確実性の高い金融・ギャンブル市場の価格形成プロセスには、行動経済学の分野で証明されている **「本命・大穴バイアス (Favorite-Longshot Bias)」** と呼ばれる現象が存在します。

これは以下のような人間固有の認知バイアスです：
* 「人々は当たる確率の極めて低い大穴馬に対して『一攫千金の夢』を過剰に見て資金を投じるため、大穴馬のオッズは実質的な期待勝率から算出される理論オッズよりも不当に下がる（売れすぎる）」
* 「逆に、極めて高い確率で勝つとされる本命馬については、『当たってもリターンが少なすぎるため面白くない』と敬遠されがちになり、結果として本命馬のオッズは理論オッズよりも僅かに上がる（過小評価される）」

モデルが出力する「その馬の持つ純粋なポテンシャルに基づく勝率（それに沿ったフェアな支持率）」と「大衆が実際に作る支持率＝オッズ」の間には、このバイアスによるズレが生じます。
これを補正調整するため、対数領域から戻した生確率推論値 $\hat{P}\_{raw}$ に対して、累乗パラメータ $k$ を導入する「キャリブレーション・レイヤー（補正層）」を最終出力段に組み込みました。

$$ \hat{P}\_{calibrated} = \hat{P}\_{raw}^k $$

* **$k = 1.0$** : 補正なし（非バイアス状態の純粋な支持率）。
* **$k > 1.0$** : 生の確率が高いもの（本命）の確率をさらに押し上げ、確率が低いもの（大穴）をさらに押し下げる写像変換。これは「ファンの評価不足（本命敬遠・穴党）」を補正する。
* **$k < 1.0$** : 確率を均等に近づける変換（大穴を過大評価するファンの心理状態をシミュレーション）。
